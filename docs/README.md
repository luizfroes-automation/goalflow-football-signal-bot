# GoalFlow - Football Signal System

GoalFlow is a live football signal system designed to detect goal-scoring opportunities using real-time match data.

This project was built using workflow automation with n8n and integrates external APIs, messaging systems, and data storage for analysis.

---

## 📌 Overview

The system collects live football data, processes match statistics, applies rule-based logic, and generates signals when specific conditions are met.

Signals are sent via Telegram and stored for further analysis and performance evaluation.

---

## 🎯 Objectives

- Learn workflow automation using n8n  
- Integrate real-time football data APIs  
- Build a rule-based decision system  
- Send automated alerts via Telegram  
- Store signal data in Google Sheets  
- Analyse signal performance with a dashboard  

---

## ⚙️ Tech Stack

- n8n (workflow automation)  
- Football Data API  
- Telegram Bot  
- Google Sheets  

---

## 🔄 System Workflow

1. Fetch live football match data from API  
2. Process and filter match statistics  
3. Apply signal detection rules  
4. Send alert via Telegram  
5. Store signal data in Google Sheets  
6. Analyse results and accuracy
